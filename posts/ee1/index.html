计算机系统基础知识

计算机的基本硬件系统由运算器，控制器，存储器，输入输出设备组成

  1. 运算器，控制器被集成为中央处理单元（CPU），是硬件系统的核心
  2. 存储器，分为内存和外存。内存速度高，容量小，临时存储。后者容量大，存储慢，长期存储
  3. 输入和输出设备，统称为外设

CPU的功能

  1. 程序控制
    CPU通过指令来控制程序
  2. 操作控制
    一条指令执行需要若干操作信号配合完成
  3. 时间控制
    指令执行过程中操作信号的出现时间，持续时间以及出现的时间顺序需要进行严格控制
  4. 数据处理
    对数据的加工时CPU的最基本的任务

运算器的组成

  1. 算数逻辑单元 ALU ：实现对数据的算数和逻辑运算
  2. 累加寄存器 AC ：运算结果或者操作数的存放区
  3. 数据缓冲寄存器 DR ：暂时存放内存的指令或者数据
  4. 状态条件寄存器 PSW ：保存指令运行的条件码内容，如溢出标志

控制器的组成

  1. 指令寄存器 IR ：暂存CPU执行指令
  2. 程序计数器 PC ：存放指令地址
  3. 地址寄存器 AR ：保存当前CPU所访问的内存地址
  4. 指令译码器 ID ：分析指令操作码

进制的转化

  1. R进制转为十进制
    位权展开法
  2. 十进制转为R进制
    十进制处以R取余数
  3. 十进制小数
    乘以R取整数
  4. m进制转为n进制
    先将m进制转为十进制，十进制转为n进制

数的表示

各种数值在计算机中的表示称为机器数，特点是使用二进制计数制，数的符号用0和1表示，小数点则隐含，不占位置
机器数有有符号和无符号之分，无符号表示正数，没有符号位。带符号最高位为符号位，正数符号位为0，负数为1

  1. 纯小数
    约定小数点的位置在机器数的最高数值位之前
  2. 纯整数
    约定小数点的位置在机器数的最低数值位之后
  3. 真值
    机器数对应的实际数值
    带符号数的编码方式
  4. 原码
    一个数的正常二进制表示，最高位表示符号。数值0的源码有两种形式+0和-0
  5. 反码
    正数的反码即原码，负数的反码为符号位之外均取反
  6. 补码
    正数的补码即原码，负数的补码是反码末尾加一
  7. 移码
    补码首位取反，用作浮点运算

浮点数运算

  1. 对阶
    小阶对大阶
  2. 尾数运算
  3. 结果规格化
    尾数表示规格化（1.0xxxx 0.1xxxx）

校验码

码距：从A转为B所需要改变的位数称为码距，码距越大，越利于纠错

  1. 奇偶校验码
    增加一位校验位使1的个数为奇数 奇校验 或者偶数 偶校验。可以检查错误，无法改错
  2. 循环冗余校验码CRC
    只能检错，不能纠错，找出一个能整除多项式的编码，首先将原始报文除以多项式，将所得余数作为校验位加在原始报文之后
    即传输一个可以被多项式整除的信息，在另一侧除以多项式，看余数大小
    先在原始信息后面添加0，位数等于多项式阶数，作为被除数–》多项式转除数，幂指数存在为1，否则为0
  3. 海明校验码
    本质也是奇偶校验，2的n次方都是校验位
